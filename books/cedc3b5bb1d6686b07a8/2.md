---
title: "開発必ず使うコマンドを叩く"
---
Gitにはたくさんのコマンドとオプションがありますが、その中でも開発で必ず使うコマンドをまとめています。

コマンドを列挙する

| コマンド                               | 内容                            |
| ---------------------------------- | ----------------------------- |
| `git status`                       | 前回コミット分から差分があるファイルを表示する       |
| `git add <filename>`               | filenameのファイル、ディレクトリをステージングする |
| `git commit -m '<commit message>'` | ステージングしたファイルを<commit message>でコミットする |
|                                    |                               |
